# Отчет по ЛР-1: Класс и инкапсуляция

Реализована сущность «Пациент» для предметной области «Медицина».

## Процесс и ответы на вопросы

### Вопрос 1. Что является сущностью?
Сущностью является **Patient** (Пациент). Это реальный объект медицинской системы, обладающий персональными данными, состоянием здоровья и финансовым балансом для оплаты услуг

### Вопрос 2. Какие у него атрибуты?
Каждый атрибут в классе `Patient` выбран не случайно и имеет свою роль:
1.  **`_name` (Имя)** — нужен для идентификации пациента. В этой работе он неизменяем (устанавливается один раз при создании)
2.  **`_age` (Возраст)** — критический атрибут для медицины. Он может **изменяться** (через сеттер), если пациент отмечает день рождения, но всегда проходит проверку на вхождение в диапазон [0, 120]
3.  **`_balance` (Баланс)** — финансовый показатель. Он **уменьшается** на 100 единиц при вызове метода лечения (`treat`), если пациент болен. Мы не можем лечить "в долг" (инвариант баланс >= 0)
4.  **`_status` (Здоровье)** — логическое состояние. Оно **меняется** с помощью методов `get_sick()` (статус становится "болен") и `treat()` (статус возвращается в "здоров")

### Вопрос 3. Какие инварианты?
Инварианты — это правила, которые никогда не должны нарушаться:
- Имя не может быть пустой строкой
- Возраст должен быть целым числом в диапазоне от 0 до 120
- Баланс не может быть отрицательным числом
- Статус принимает только определенные логические значения ("здоров", "болен")

### Вопрос 4. Что значит “равенство”?
Два объекта класса `Patient` считаются равными, если у них полностью совпадают **имя** и **возраст**. Баланс и текущее состояние здоровья при этом могут отличаться (один и тот же человек может прийти в больницу с разными деньгами или в разное время)

### Вопрос 5. Есть ли состояние?
Да, у пациента есть состояние здоровья (`status`)
Правила изменения состояния:
- Нельзя «вылечить» пациента, если он уже в статусе «здоров»
- Нельзя провести лечение, если на балансе недостаточно средств (стоимость лечения — 100 у.е.)
- При успешном лечении статус меняется на «здоров», а баланс уменьшается
