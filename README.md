# Лабораторные работы по ООП (Python)

## Лабораторная работа №1: Класс и инкапсуляция

**Вариант:** 10 (Предметная область: Медицина / Пациент)

### Описание
В рамках первой лабораторной работы реализован класс `Patient` (Пациент), демонстрирующий основные принципы ООП:
- Создание классов и объектов.
- **Инкапсуляция** (защита данных с помощью `_`).
- **Валидация** атрибутов при создании и изменении.
- Использование свойств (**`@property`** и сеттеров).
- Переопределение **магических методов** (`__str__`, `__repr__`, `__eq__`).
- Реализация **бизнес-логики** и состояний объекта (методы `get_sick` и `treat`).

### Структура проекта
- `src/lab01/model.py` — реализация класса `Patient` со всеми проверками и логикой.
- `src/lab01/demo.py` — демонстрационный скрипт, показывающий работу класса на понятных примерах (создание, ошибки, лечение).

### Как запустить демо-сценарий
Находясь в корневой папке проекта, выполните команду в терминале:
```bash
python src/lab01/demo.py
```
